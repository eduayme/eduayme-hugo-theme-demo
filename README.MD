EDUAYME Hugo Theme by <a href="https://github.com/eduayme">@eduayme</a>
</h1>

## Installation
```
$ brew install hugo
$ hugo new site blog-eduayme
$ cd blog-eduayme
$ git init
$ git submodule add git@github.com:eduayme/eduayme-hugo-theme.git themes/eduayme
$ echo 'theme = "eduayme"' >> config.toml
```
